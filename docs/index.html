Quick Start

Up and running in 60 seconds

No npm. No bundler. No config.

Just one script tag and your first ai-* class.

1

Add the script tag to your page

HTML 
<script src="ai-automation-sdk.min.js"></script>

Place before your closing </body> tag. The SDK auto-initialises and adds a floating mic button.

2

Add ai-* classes to your elements

HTML 
<!-- Voice-fill your contact form -->
<form class="ai-voice-fill">
  <input name="name" placeholder="Full name" />
  <input name="email" placeholder="Email" />
</form>

<!-- Voice-controlled scrolling -->
<body class="ai-smart-scroll ai-navigate">
3

Click the mic button and speak

The floating 🎙 button in the bottom-right corner starts listening. Grant microphone permission when prompted.

🎤 Try saying "fill name John Smith", "scroll down", or "search for AI tools"
🎙

Module

Voice Form Fill

Enable hands-free form filling on any <form>. On activation the SDK automatically scrapes the form to detect all fields (by name, placeholder, aria-label, associated <label>) and shows a visual detection panel. Say "autofill" to fill them all step-by-step.

class="ai-voice-fill"

Voice Commands

Say Action
"what fields" Shows a list of all detected fields
"autofill" Starts a sequential guided session — fills each field one by one
"fill name John Smith" Fills the Name field directly
"fill email" Selects Email field, awaits your next utterance as the value
"skip" / "next field" Skips the current field during autofill session
"submit" Submits the form
"clear form" Clears all inputs
"stop filling" Cancels the current session

Usage

HTML 
<form class="ai-voice-fill">
  <input type="text"  name="name"    placeholder="Full name"    />
  <input type="email" name="email"   placeholder="Email"         />
  <input type="tel"   name="phone"   placeholder="Phone number"   />
  <button type="submit">Submit</button>
</form>

Live Demo

Module

Smart Scroll

Navigate any page by voice. Knows about your page's headings and sections — so "next section" actually jumps to the next heading intelligently.

class="ai-smart-scroll"

Voice Commands

Say Action
"scroll down" Scroll down 40% of viewport
"scroll up" Scroll up 40% of viewport
"scroll down a lot" Scroll down 80% of viewport
"page down" Scroll one full page
"scroll to top" / "go to top" Jump to page top
"scroll to bottom" Jump to page bottom
"next section" Jump to the next heading/section
"previous section" Jump to the previous heading/section

Usage

HTML 
<!-- Apply to body for page-wide scroll -->
<body class="ai-smart-scroll">

<!-- Or apply to a scrollable container -->
<div class="ai-smart-scroll" style="height:400px; overflow-y:auto">
  ...
</div>

Module

Search Automation

Add ai-search to any search input. Voice commands populate the field and trigger form submission instantly.

class="ai-search"

Voice Commands

Say Action
"search for [query]" Fill input + submit form
"find [query]" Fill input + submit form
"look up [query]" Fill input + submit form
"clear search" Empty the search input

Usage

HTML 
<form action="/search">
  <input
    type="search"
    class="ai-search"
    name="q"
    placeholder='Try: "search for JavaScript tutorials"'
  />
</form>

Module

Navigation

Control browser navigation by voice. Navigate back, forward, reload, jump to any URL or anchor, or open new tabs — all hands-free.

class="ai-navigate"

Voice Commands

Say Action
"go back" history.back()
"go forward" history.forward()
"refresh" location.reload()
"go home" Navigate to /
"open github.com" Navigate to github.com
"go to dashboard" Navigate to closest matching URL or Google search
"open new tab wikipedia.org" Open URL in a new tab
"scroll to #contact" Smooth-scroll to element with id contact

Usage

HTML 
<!-- Apply to body for global navigation commands -->
<body class="ai-navigate ai-smart-scroll">

Reference

All Voice Commands

Complete reference for all voice commands across all modules.

Module Command Action
voice-fill "what fields" Show all detected fields
voice-fill "autofill" Sequential guided fill session
voice-fill "fill [field] [value]" Fill named field with value
voice-fill "fill [field]" Select field, await value
voice-fill "skip" / "next field" Skip current session field
voice-fill "submit" Submit active form
voice-fill "clear form" Clear all form fields
smart-scroll "scroll down/up" Scroll 40% of viewport
smart-scroll "scroll down a lot" Scroll 80% of viewport
smart-scroll "page down/up" Scroll full viewport height
smart-scroll "scroll to top/bottom" Jump to page top/bottom
smart-scroll "next/previous section" Jump between headings
search "search for [query]" Fill + submit search
search "find [query]" Fill + submit search
search "clear search" Clear search input
navigate "go back/forward" Browser history
navigate "refresh" Reload page
navigate "go home" Navigate to /
navigate "open [url]" Navigate to URL
navigate "open new tab [url]" Open URL in new tab
navigate "scroll to #[id]" Jump to element

Compatibility

Browser Support

AI Automation SDK uses the Web Speech API which requires a modern Chromium-based browser.

Browser Voice Commands Notes
✅ Chrome 25+ Full support Recommended
✅ Edge 79+ Full support Chromium-based
✅ Opera 27+ Full support Chromium-based
⚠️ Safari 14.1+ Partial Requires user gesture; continuous recognition limited
❌ Firefox Not supported No Web Speech API. SDK loads without errors but mic is disabled.
💡
The SDK gracefully degrades — if the browser doesn't support speech recognition, the mic button is hidden and all other page functionality works normally.
